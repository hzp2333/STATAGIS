In the package, .\1.step1_Figure3.py, .\2.step2.py and .\3.step3.do are the code that contain the code for the estimation of efficiency tree and the data required for the subsequent figures and tables.

  • We list how the data is produced, and mark the number of lines corresponding to the input data in the code.

  • The root directory is efficiency_tree, which is replaced by ..

  • .\city_inefficiency_tree.xlsx, .\temp\fig1Pre.xlsx, .\step2Pre.xlsx and .\chinamap\chinacity40_db.dta are the original data, see the manuscript for the data source.
